Quick Start
----------------------------------------
Create a codespace.  You'll see a web-based version of VS Code connected to a  Linux Machine.

<img src="screenshot.png" width=300>

Compile and run using the VS Code Terminal.
```console
$ make
$ ./main
Hello World!
```
